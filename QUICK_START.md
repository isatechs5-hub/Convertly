## Troubleshooting

### If a feature doesn't work:
1. Check file format is correct (.png, .pdf, etc.)
2. Try a smaller file (< 10MB recommended)
3. Check browser console for errors (F12)
4. Refresh the page and try again

### Common Issues:
- **"File too large"**: Compress your file first
- **"Conversion failed"**: File might be corrupted
- **"Download not starting"**: Check browser download settings
